# 회의록 (2026-04-02)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 전처리 재검증 및 피드백 분석

## 2. 금주 진행 사항

### 2.1 전처리 무결성 재검증
- 신규 병합된 데이터에 지난주 구축한 전처리 파이프라인을 동일하게 적용하여 추가적인 이상치가 존재하지 않음을 확인함.
- 합금 성분의 결측치는 미첨가(0)로 대체함으로써 데이터의 통계 무결성을 유지함.

### 2.2 합금 조성 기반 데이터 그룹화 및 Train/Test 분할
- 특정 성분비에 대한 모델의 과적합(Overfitting)을 방지하고 범용적인 예측 성능을 검증하기 위해 데이터 분할 기준을 재정립함.
- 동일 제품에 대한 다수의 측정치가 존재하는 데이터 특성을 고려하여, 제품(합금 조성) 단위를 기준으로 Train set과 Test set을 그룹으로 분리함.
- 이를 통해 학습되지 않은 새로운 제품군에 대해서도 신뢰도 높은 수명 예측이 가능하도록 평가 기반을 마련함.

### 2.3 이전 발표 피드백 반영 및 외부 데이터셋 추가 확보
- 지난 프로젝트 발표에서 수렴된 피드백 사항들을 종합적으로 분석하여 모델링 전략을 개선함.
- 앞서 적용한 제품 단위 데이터셋 분할과 마찬가지로 해당 피드백을 반영함.
- 학습 데이터의 양적 한계를 보완하고 모델의 일반화 성능을 높이기 위해 K-MDS 및 MatNavi 데이터셋을 추가로 확보하여 향후 모델 성능을 높히기 위해 데이터 방향성을 구체화함.

## 3. 주요 의사결정 사항
- 데이터 분할 기준을 제품 단위 그룹화로 확정하여 일반화 성능 검증의 신뢰도를 강화함.
- 합금 성분 결측치 처리(미첨가=0) 기준을 전처리 표준안으로 유지함.
- 외부 데이터셋(K-MDS, MatNavi)을 추가로 확보하여 모델 학습 데이터 확대 계획을 수립함.

## 4. 리스크 및 대응
- 리스크 1: 외부 데이터셋의 스키마 및 단위 불일치
  - 대응: 기존 프로젝트 스키마에 맞게 변수명 및 단위를 표준화하여 통합함.
- 리스크 2: 이종 데이터 통합 시 데이터 품질 편차
  - 대응: 통합된 신규 데이터에 기존 전처리 파이프라인을 일괄 적용하여 무결성 확보.
- 리스크 3: 제품 단위 분할로 인한 학습 데이터 부족
  - 대응: 외부 데이터셋 확보를 통해 학습 데이터 볼륨 확대로 대응.

## 5. 차주 계획

### 5.1 K-MDS 및 MatNavi 등 외부 데이터셋 확보 및 병합
- 공신력 있는 외부 소재 데이터베이스인 K-MDS와 MatNavi 데이터셋을 수집할 예정.
- 수집된 이기종 데이터의 변수명 및 단위를 기존 프로젝트 스키마에 맞게 표준화하여 통합 데이터셋으로 병합하는 작업을 수행할 예정.

### 5.2 신규 데이터 전처리 파이프라인 적용 및 가공
- 추가 확보된 신규 데이터에 기존에 구축해 둔 전처리 파이프라인을 일괄 적용함.
- 결측치 및 이상치 처리를 수행하고 물리 기반 피처(가혹도 지수 및 LMP 등)를 동일하게 산출하여 데이터의 물리적 무결성을 확보함.

## 6. 기대 산출물

- 신규 데이터 전처리 적용 결과 및 무결성 검증 리포트
- 외부 데이터 통합 후 제품 단위 분할 재구성 현황

## 7. 결론
- 금주에는 기존 전처리 파이프라인의 안정성을 재검증하고, 제품 단위 데이터 분할을 통해 일반화 성능 검증 체계를 고도화함.
- 외부 데이터셋 추가 확보를 통해 학습 데이터의 양적 한계를 해소하고, 다음 단계의 모델 성능 향상을 위한 토대를 마련함.
- 다음 주는 데이터의 표준화 및 통합을 정확하게 수행하여 모델 학습의 신뢰도를 보장하는 것이 핵심 과제임.
