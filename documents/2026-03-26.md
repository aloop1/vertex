# 회의록 (2026-03-26)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 물리 기반 피처 확장 결과 점검 및 검증 전략 고도화 방향 확정

## 2. 금주 진행 사항

### 2.1 물리 기반 피처 엔지니어링 및 차원 확장
- 고온·고압 환경에서의 비선형 거동을 반영하기 위해 Hollomon-Jaffe 파라미터를 응용한 가혹도 지수(Severity Index) 3종을 신규 생성함.
- 온도와 시간의 상호작용을 물리 공식으로 수치화하여 기존 31개에서 총 42개로 피처를 확장함.
- 단순 통계 학습이 아닌 물리 법칙 기반 패턴 인식을 강화하는 방향으로 입력 공간을 재설계함.

### 2.2 통계적 데이터 정제 및 무결성 확보
- 합금 성분 결측치는 공정 의미를 반영해 '미첨가(0)'로 일괄 대체함.
- 음수 온도, 0 이하 수명 등 물리적으로 불가능한 값을 필터링하는 전처리 파이프라인을 구축함.
- Stress 변수 내 약 14% 이상치 후보를 검토한 결과, 사전 정의한 입력 범위(5~450MPa) 내 정상값으로 확인되어 제거 없이 학습 데이터로 유지함.

### 2.3 기술 자문 및 변수 전략 논의
- 전처리 파이프라인과 히트맵 분석 결과를 기반으로 장재훈 교수님과 기술 자문을 진행함.
- 오스테나이트계 합금의 유효 변수 선별 기준과 제품군 단위 분할 방식의 타당성을 중점 논의함.

## 3. 주요 의사결정 사항
- 물리 기반 파생변수(가혹도 지수)를 포함한 확장 피처 체계를 유지하고, 추가 물리 지수 도입을 검토함.
- 결측치/비정상값 처리 기준(미첨가=0, 물리 불가능값 제거)을 전처리 표준안으로 채택함.
- 응력 이상치로 분류된 값은 도메인 입력 범위 검증 결과에 따라 제거하지 않고 유지함.
- 검증 전략은 임의 샘플 분할을 넘어 제품군 단위 분할 방식으로 고도화함.

## 4. 리스크 및 대응
- 리스크 1: 파생 피처 증가에 따른 모델 복잡도 상승 및 과적합 가능성
  - 대응: 변수 중요도 기반 선택, 저기여 피처 제거, 분할 검증 강화
- 리스크 2: 제품군 분포 불균형으로 인한 일반화 성능 편차
  - 대응: 제품군 단위 학습/테스트 분리 로직 적용 및 군별 오차 비교
- 리스크 3: 물리 지수 해석의 불확실성
  - 대응: LMP 추가 산출 후 분포 시각화 및 성능 기여도 재검증

## 5. 차주 계획
- 가혹도 지수 외 라르손-밀러 파라미터(LMP)를 추가 산출하고, 분포 편차 시각화를 통해 유효성을 재검증함.
- 오스테나이트계 합금에서 수명 영향이 낮은 변수(예: 냉각 방식)를 재평가하여 삭제 여부를 확정함.
- 학습 데이터에 포함되지 않은 신규 제품군을 테스트 셋으로 구성하는 제품군 단위 재분할 로직을 설계·구현함.
- 재분할 데이터 기준으로 모델을 재학습하고 실전 예측 성능을 점검함.

## 6. 기대 산출물
- 물리 기반 파생지수(Severity Index, LMP) 검증 메모 및 분포 시각화 결과
- 변수 최적화 반영 피처셋 정의서
- 제품군 단위 분할 검증 결과표 및 재학습 성능 리포트

## 7. 결론
- 금주에는 물리 기반 피처 확장과 데이터 무결성 확보를 통해 모델 입력 신뢰도를 높였고, 도메인 자문을 통해 변수·검증 전략의 실무 적용성을 구체화함.
- 다음 단계의 핵심은 제품군 단위 검증 체계를 정착시키고, 추가 물리 지수의 실효성을 정량적으로 확인해 일반화 성능을 안정화하는 것임.
