# 회의록 (2026-05-21)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 증강 데이터 검증, 최적화 기법 적용, UI/UX 개선 현황 점검

## 2. 금주 진행 사항

### 2.1 증강 데이터 성능 검증 및 앙상블 적용 검토
- 원본 데이터 기반 모델과 LMP 기반 증강 데이터로 학습한 모델의 성능을 비교 분석함.
- 학습 데이터가 2,682건에서 3,843건으로 약 43% 증가했으며, 로그 스케일 기준 R², RMSE, MAE 등 핵심 지표가 약 10% 향상됨.

### 2.2 UI/UX 대시보드 기능 다양화 및 속도 최적화
- Streamlit 환경에서 히트맵 렌더링 속도를 최적화하고 사용자 편의성을 고려한 분석 기능을 추가 구현함.
- 레이아웃 개선을 통해 주요 합금 원소별 수명 변화를 직관적으로 비교할 수 있는 대조 분석 기능을 도입함.

### 2.3 그룹별 LMP 피팅 시각화 및 데이터 I/O 체계화
- 그룹별 LMP 피팅 그래프를 시각화하여 모델 적합도를 점검함.
- 합성 데이터와 모델 출력 파일을 지정된 로컬 경로에 일관되게 저장하는 입출력 체계를 안정화함.
- 가독성 높은 테이블 뷰를 추가해 결과 확인 편의성을 개선함.

### 2.4 유전 알고리즘(GA) 최적화 구현
- 원소 탐색 공간 정의, 물리 기반 패널티, OOD 패널티 구조를 설계함.
- NSGA-II 기반 다목적 최적화 및 합금 총합 제한(repair) 로직을 구현하고, 결과를 JSON/CSV로 출력하도록 구성함.

## 3. 주요 의사결정 사항
- LMP 증강 데이터의 성능 향상이 확인되어 앙상블 적용을 본격 검토하기로 결정함.
- Streamlit 대시보드의 시각화 성능을 우선 개선하여 사용자 피드백을 수집하기로 함.
- GA 최적화 결과에 대해 pycalphad 기반 상평형 검증을 의무화하도록 합의함.

## 4. 리스크 및 대응
- 리스크 1: 증강 데이터로 인한 과적합 또는 물리적 비타당성
  - 대응: 증강 데이터 분포 비교 및 도메인 전문가 검토, OOD 패널티 적용
- 리스크 2: 대시보드 렌더링 병목
  - 대응: 시각화용 데이터 샘플링 및 서버 사이드 캐싱 도입
- 리스크 3: 유전 알고리즘의 계산 비용 및 지역 최적해
  - 대응: 탐색 공간 제한, 병렬화 및 다목적 최적화(NSGA-II) 적용

## 5. 차주 계획

### 5.1 유전 알고리즘 결과 검증 및 후보 조성 분석 고도화
- pycalphad 기반 상평형 후처리 검증을 수행하고 OOD 패널티 구조를 고도화함.

### 5.2 Transformer-트리 앙상블 커스텀 모델 구조 및 성능 개선
- Transformer와 트리 기반 모델의 잔차 보정 가중치를 재조정하여 안정성을 향상시키고 극한 조건에서의 불안정성 문제를 해결할 개선안을 검토함.

### 5.3 시스템 통합 테스트 및 UI/UX 최종 보완
- 최적화된 GA 엔진을 Streamlit 대시보드에 연동하고 데이터 입출력 시 진행 상태를 시각적으로 표시하는 기능을 추가함.

## 6. 기대 산출물
- GA 최적화 결과 및 pycalphad 검증 리포트
- Transformer-트리 앙상블 성능 비교 보고서
- Streamlit 통합 대시보드

## 7. 결론
- 증강 데이터와 최적화 기법을 병행 적용하여 모델 성능을 개선하고, UI/UX 개선을 통해 사용성 및 검증 편의성을 높이는 방향으로 진행함.
- 다음 주에는 GA 결과의 물리적 검증과 앙상블 모델 안정화에 우선적으로 집중할 예정임.
