# 회의록 (2026-05-07)

## 1. 회의 개요
- 프로젝트명: 소재부품 수명 예측 모델 개발
- 회의 목적: 데이터 증강, 시각화 전략 수립

## 2. 금주 진행 사항

### 2.1 중간발표 피드백 분석 및 도메인 지식 학습 계획 수립
- 재료공학적 이해도 제고: 크립 데이터의 물리적 경향성(온도·응력·시간의 상관관계)을 분석 모델에 심도 있게 반영하기 위해 관련 재료공학 논문 분석 등 추가 학습 및 반영 계획을 수립함.
- 미지 조성 예측 논리 정교화: 전통적 플로팅 방식은 기존 조성에 한정적이므로 성분이 다른 신규 합금을 예측할 수 있는 AI 모델의 논리적 우위 및 필요성을 재확인함.

### 2.2 데이터 표준화 및 증강 전략 수립
- 크립 데이터의 수집 한계(양적 부족, 빈 공간)를 해결하기 위해 물리적 거동(LMP 등)을 반영한 데이터 증강 필요성을 확인함.
- 데이터를 단일 칼럼이 아닌 물리적 흐름(이미지, 시퀀스)으로 인식시키는 학습 구조를 검토함.

### 2.3 결과물 시각화 및 패키징 전략 논의
- 직관적 시각화: 온도·응력 변화에 따른 수명 분포를 나타내는 2D/3D 컬러 맵 구현 방안 검토.
- 임팩트 중심 인터페이스: 크립 파단 과정을 시각적으로 표현하는 애니메이션을 포함한 합금 분석 리포트 자동 생성 기능 도입을 논의함.

## 3. 주요 의사결정 사항
- 물리 법칙 기반 데이터 증강 연구를 우선 과제로 설정함.
- SHAP 분석 및 LMP/마스터 커브 기반 검증을 통해 모델 설명력과 물리적 타당성을 병행 검증하기로 결정함.
- 기존 Streamlit 대시보드에 온도·응력별 수명 히트맵을 우선 추가하기로 합의함.

## 4. 리스크 및 대응
- 리스크 1: 증강 데이터의 물리적 타당성 부족
  - 대응: 물리 기반 생성 규칙(LMP) 적용 및 도메인 검토 수행
- 리스크 2: 시각화 구현 시 문제
  - 대응: 전처리 단계에서 요약 통계/샘플링으로 시각화용 데이터 경량화
- 리스크 3: 미지 조성 예측의 불확실성
  - 대응: 논문·전문가 검토를 통한 모델 제약 조건 정의 및 보수적 예측 범위 표기

## 5. 차주 계획

### 5.1 데이터 증강 및 물리 거동 학습
- 물리 법칙 기반 가상 데이터 생성 파이프라인 파일럿 구현
- 생성 데이터의 분포 비교 및 유효성 검증 수행

### 5.2 SHAP 및 LMP 커브 기반 예측 검증
- SHAP 분석을 통해 원소별 기여도 파악
- 마스터 커브·LMP 기반 모델 타당성 검증 프로토콜 수립

### 5.3 Streamlit 대시보드 시각화 고도화
- 온도·응력별 수명 히트맵 추가 및 애니메이션 샘플 구현
- 시각화 파이프라인 성능·응답성 최적화

## 6. 기대 산출물
- 물리 기반 데이터 증강 파일럿 및 검증 리포트
- SHAP 분석 결과 및 LMP 커브 비교 문서
- Streamlit 히트맵, 애니메이션 시연 샘플

## 7. 결론
- 중간발표 피드백을 반영해 데이터 증강과 물리 기반 검증을 병행함으로써 모델의 일반화 및 설명력을 강화할 계획임.
- 다음 주에는 가상 데이터 생성 파이프라인 파일럿과 SHAP/LMP 검증 작업을 우선 진행함.
